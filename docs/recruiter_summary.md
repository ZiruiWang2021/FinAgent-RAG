# Recruiter Summary / 项目摘要

FinAgent-RAG is a compact AI engineering portfolio project for financial document analysis.  
It combines a FastAPI backend, React TypeScript frontend, document RAG, stock-data tool calls, and a simple multi-agent workflow.  
Users can upload reports, ask source-grounded questions, inspect citations, view agent steps, review tool calls, and generate structured reports.  
The project includes Pydantic schemas for typed API responses and structured report validation.  
It also includes a lightweight evaluation module with fixed questions, expected answer points, groundedness scoring, and an unknown-answer case for hallucination control.  
The implementation is intentionally small and readable, making it suitable for discussing AI application architecture, tradeoffs, and next-step improvements.  
It does not claim to be a production trading or investment system; it is a runnable prototype focused on practical AI engineering patterns.

中文摘要：

FinAgent-RAG 是一个面向金融文档分析的紧凑型 AI 工程作品集项目。  
它包含 FastAPI 后端、React TypeScript 前端、文档 RAG、股票数据工具调用和一个简单的多 Agent 工作流。  
用户可以上传研报或公告，进行带来源引用的问答，查看 Agent 步骤、工具调用记录，并生成结构化报告。  
项目使用 Pydantic schema 校验 API 响应和结构化报告输出。  
它还包含轻量评估模块，通过固定问题、期望答案要点、groundedness score 和未知问题用例检查基础 hallucination 风险。  
实现保持小而清晰，适合讨论 AI 应用架构、工程取舍和后续扩展方向。  
它不声称是生产级交易或投资系统，而是聚焦实际 AI 工程模式的可运行原型。
